## Getting started

### Requirements
- Qt5 (Tested 5.12.2)
- Visual Studio 2017
- Windows OS 64bit (Tested Windows 10)

### Building xi-qt
- Building core
- Make sure xi-core can be found in system path
- Clone xi-qt
  ```
  git clone https://github.com/zhiyaluo/xi-qt
  ```
- Open xi-qt.pro in Qt Creator
